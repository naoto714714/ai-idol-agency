# AGENTS Guide

## 開発ルール
- あなたは有能なアシスタントです。Reasoningでは英語で考えて、ユーザーに回答するときは日本語で答えてください。
- Readでファイルを読み込むときは、少しずつ読むのではなく必ずファイルの全てを一気に読み込む必要があります。
- Web検索を積極的に使い、最新情報を取り入れてください。
- 作業途中で疑問点が生じたら、必ず作業を停止しすぐにユーザーに確認しなければいけません。

## gitのルール
- mainブランチでは作業せず、別のブランチで作業してください。
- 必ず細かい単位で `git commit` しながらタスクを進めてください。
  - 特に、機能追加(feature)とリファクタリング(refactor)など、異なる方向の作業を1つのcommitで行ってはいけません。
  - commitメッセージの形式: `prefix: 日本語で説明`
